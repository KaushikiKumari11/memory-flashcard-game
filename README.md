# Memory Flashcard Game

A simple and interactive memory card matching game built using HTML, CSS, and JavaScript. The game tests a player's memory by requiring them to match pairs of cards within a grid.

# Features

- Card flip animation on click  
- Matching logic to check for identical pairs  
- Reset functionality to restart the game  
- Responsive design for different screen sizes  
- Simple and user-friendly interface  

# Technologies Used

- HTML  
- CSS  
- JavaScript  

# How It Works

1. The game displays a grid of face-down cards  
2. The player clicks on two cards to flip them  
3. If the cards match, they remain flipped  
4. If they do not match, they flip back after a short delay  
5. The game continues until all pairs are matched
